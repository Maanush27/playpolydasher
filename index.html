How to program in C#

Grammar

C# code is based on certain rules.

  • C# is capital based
  • End statements with semicolons (;)
  • Code blocks such as if/else statements and loops are in curly brackets ({})

Data types

Data types are types of values C# can understand. Here are some of the basic data types:

  • Int: Integers are numbers that do not contain decimals
  • Float: Floats are numbers that contain decimals
  • String: Strings are text used with quotations
  • Boolean: Booleans are statements that can only have 2 values: true or false

Variables

Variables are names that store values of any data type. To use a variable you write the data type before declaring the name of the variable. For example:

  • public int myNumber = 67;
  • public float myDecimalNumber = 14.32;
  • public string mySentence = "Hello, my name is John"
  • public boolean isCSharpCapitalBased = true;
